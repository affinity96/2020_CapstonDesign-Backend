# HOMEKIPPA_BACKEND

실행방법

1. node js, npm 설치
2. git clone
3. git checkout -v '브랜치 이름'  :  본인 브랜치 생성 및 이동
3. npm install  :  모듈 설치
4. node app.js   :  서버 시작
